# Модуль Математика (`математика`)

Модуль математики содержит расширенные математические функции. Чтобы его использовать, напишите `модуль математика` в начале файла.

## Список функций

* `математика.случайное(мин, макс)`
Возвращает случайное целое число в диапазоне от `мин` до `макс` включительно. Это одна из самых важных функций для создания игр!
```son
пусть кубик = математика.случайное(1, 6)
напечатай "Выпало: " + строка(кубик)
```

* `математика.корень(число)`
Возвращает квадратный корень числа.
```son
напечатай математика.корень(16) # 4
```

* `математика.синус(число)` / `математика.косинус(число)`
Тригонометрические функции (принимают радианы).

* `математика.пол(число)` / `математика.потолок(число)`
Округление дробных чисел:
`пол` округляет всегда вниз (отбрасывает дробную часть).
`потолок` округляет всегда вверх до целого.
```son
напечатай математика.пол(3.8) # 3
напечатай математика.потолок(3.1) # 4
```

* `математика.abs(число)`
Возвращает модуль числа (всегда положительное).
```son
напечатай математика.abs(-50) # 50
```

* `математика.pow(основание, степень)`
Возводит число в заданную степень.
```son
напечатай математика.pow(2, 3) # 2 в степени 3 = 8
```
