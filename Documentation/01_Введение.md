# Введение в SonLang

Добро пожаловать в официальную документацию языка **SonLang**!

SonLang — это простой, интуитивно понятный язык программирования на русском языке, созданный для того, чтобы код читался как обычный текст. Язык отлично подходит для новичков, написания системных скриптов, а также для создания графических интерфейсов (GUI) и 2D-игр.

## Философия языка

1. **Естественность**: Код пишется кириллицей, а названия встроенных функций и операторов звучат как обычные русские слова (`если`, `пока`, `пусть`, `функция`).
2. **Отсутствие визуального мусора**: В SonLang нет обязательных точек с запятой на конце строк, а для блоков кода не требуются фигурные скобки. Блоки всегда начинаются с двоеточия (`:`) и закрываются словом `конец`.
3. **Динамическая типизация**: Вам не нужно заранее объявлять, будет ли переменная хранить текст или число. Интерпретатор сам понимает тип данных на лету.

## Установка и Запуск

### Запуск скриптов
Программы на SonLang сохраняются в файлы с расширением `.son`. 
Для запуска из командной строки используется команда `son`:

```cmd
son моя_программа.son
```

### Автоматическая интеграция
Если в папке с вашим языком вы запустили файл `install.bat`, расширение `.son` привяжется к системе. После этого вам достаточно будет **дважды кликнуть** по любому файлу `.son` в Проводнике Windows, чтобы он автоматически запустился!
