# Модуль Игра (`игра`)

Модуль `игра` создан для упрощения логики 2D-игр. Он содержит вспомогательные функции, которые берут на себя самую скучную математику.

Чтобы использовать модуль, добавьте `модуль игра` в начало файла.

## Список функций

* `игра.случайное(мин, макс)`
Возвращает случайное целое число. Идеально для спавна еды, врагов или случайных событий.
```son
пусть еда_x = игра.случайное(1, 10) * 20
```

* `игра.столкновение(x1, y1, ш1, в1, x2, y2, ш2, в2)`
Проверяет пересечение (столкновение) двух прямоугольников.
Принимает 8 параметров:
- `x1, y1` — координаты первого объекта.
- `ш1, в1` — ширина и высота первого объекта.
- `x2, y2` — координаты второго объекта.
- `ш2, в2` — ширина и высота второго объекта.

Возвращает `истина`, если объекты пересекаются, и `ложь`, если нет.

### Пример использования (поедание яблока змейкой):
```son
модуль игра

пусть змейка_x = 100
пусть змейка_y = 100
пусть размер = 20

пусть еда_x = 110
пусть еда_y = 110

если игра.столкновение(змейка_x, змейка_y, размер, размер, еда_x, еда_y, размер, размер):
    напечатай "Яблоко съедено!"
конец
```
