# Модули Время и Звук

## Модуль `время`

Этот модуль отвечает за управление ходом времени в вашей программе.
Для использования добавьте `модуль время`.

* `время.пауза(миллисекунды)`
Приостанавливает выполнение программы на указанное количество миллисекунд (1 секунда = 1000 миллисекунд).
Обычно используется в главном цикле игры, чтобы она не работала слишком быстро (например, `время.пауза(16)` дает примерно 60 кадров в секунду).
```son
напечатай "Ждем 2 секунды..."
время.пауза(2000)
напечатай "Готово!"
```

* `время.сейчас()`
Возвращает текущее системное время в секундах (Unix time - количество секунд с 1 января 1970 года).
Отлично подходит для создания тестов производительности или секундомеров.
```son
пусть старт = время.сейчас()
время.пауза(1500)
пусть финиш = время.сейчас()
напечатай "Прошло секунд: " + строка(финиш - старт)
```

## Модуль `звук`

Для использования добавьте `модуль звук`.

* `звук.гудок(частота, длительность)`
Издает простой системный звук (Beep). Работает только на операционной системе Windows.
- `частота` — высота звука в Герцах (например, 750).
- `длительность` — длина звука в миллисекундах.

```son
модуль звук
звук.гудок(1000, 500) # Гудок на частоте 1000Гц длительностью полсекунды
```

* `звук.играть("файл.wav")`
Проигрывает аудиофайл формата `.wav`. Звук воспроизводится асинхронно, то есть программа не будет ждать окончания звука и продолжит работать дальше. Идеально для фоновой музыки или звуков стрельбы/прыжков в играх.
*(Внимание: функция находится в экспериментальном режиме, убедитесь, что путь к файлу указан верно)*.
