# Условия и Циклы

Управление потоком выполнения — это способ программы принимать решения и повторять действия.

## Логические операторы
Для сравнения значений используются следующие операторы:
* `==` (равно)
* `!=` (не равно)
* `>` (больше)
* `<` (меньше)
* `>=` (больше или равно)
* `<=` (меньше или равно)

Для объединения условий:
* `и` (AND) — истинно, если оба условия истинны.
* `или` (OR) — истинно, если хотя бы одно условие истинно.
* `не` (NOT) — переворачивает значение (истина становится ложью).

## Ветвления (если / иначе)
Условия строятся с помощью ключевых слов `если`, `иначе если` и `иначе`.
Любой логический блок завершается словом `конец`. Двоеточие `:` обязательно после условия.

```son
пусть деньги = 100
пусть цена = 150

если деньги >= цена:
    напечатай "Вы купили предмет!"
    деньги = деньги - цена
иначе если деньги == 0:
    напечатай "У вас вообще нет денег."
иначе:
    напечатай "Не хватает монет: " + строка(цена - деньги)
конец
```

## Циклы

В SonLang есть два основных вида циклов. Как и условия, циклы требуют двоеточие `:` и закрываются словом `конец`.

### Цикл "пока" (while)
Выполняется до тех пор, пока условие истинно.

```son
пусть счетчик = 3
пока счетчик > 0:
    напечатай "Осталось: " + строка(счетчик)
    счетчик = счетчик - 1
конец
напечатай "Пуск!"
```

### Цикл "для" (for)
Используется для перебора чисел от начального значения до конечного.

```son
для i от 1 до 5:
    напечатай "Итерация номер " + строка(i)
конец
```

### Управление циклами (стоп и дальше)
Вы можете управлять ходом выполнения цикла изнутри:
* `стоп` (аналог break) — немедленно прерывает выполнение цикла и выходит из него.
* `дальше` (аналог continue) — пропускает оставшийся код в текущей итерации и переходит к следующему шагу цикла.

```son
пусть i = 0
пока i < 10:
    i = i + 1
    
    если i == 3:
        напечатай "Пропускаем 3"
        дальше
    конец
    
    если i == 7:
        напечатай "Хватит, выходим!"
        стоп
    конец
    
    напечатай "Число: " + строка(i)
конец
```
