# Синтаксис и Типы Данных

В этом разделе описываются базовые строительные блоки языка SonLang.

## Комментарии
Комментарии пишутся с помощью символа `#`. Всё, что идёт после решетки до конца строки, игнорируется интерпретатором.
```son
# Это комментарий. Он не будет выполнен.
пусть а = 5 # И это тоже комментарий
```

## Вывод на экран
Для вывода информации в консоль используется ключевое слово `напечатай`.
```son
напечатай "Привет, мир!"
напечатай 42
напечатай "Результат: " + строка(10)
```

## Переменные
Переменные объявляются с помощью ключевого слова `пусть`. Язык динамически определяет тип переменной по её значению.

```son
пусть имя = "Анна"       # Строка (текст)
пусть возраст = 25       # Число (целые и дробные объединяются в один тип)
пусть температура = 36.6 # Дробное число
пусть жив = истина       # Логическое значение (истина или ложь)
```

## Базовые типы данных

### 1. Числа
В SonLang числа могут быть целыми и дробными.
Доступны классические математические операторы:
* `+` (сложение)
* `-` (вычитание)
* `*` (умножение)
* `/` (деление)
* `%` (остаток от деления)

```son
пусть а = 10
пусть б = 3
напечатай а + б # Выведет 13
напечатай а / б # Выведет 3.3333333333333335
напечатай а % б # Выведет 1
```

### 2. Строки (Текст)
Строки пишутся в двойных кавычках `""`. Их можно склеивать (конкатенировать) знаком `+`.
```son
пусть город = "Москва"
напечатай "Мой город: " + город
```

У строк есть полезные встроенные методы:
* `текст.верхний()` — переводит в верхний регистр.
* `текст.нижний()` — переводит в нижний регистр.
* `текст.длина()` — возвращает длину строки.
* `текст.содержит("подстрока")` — возвращает `истина`, если текст найден.
* `текст.обрезать()` — убирает пробелы по краям.
* `текст.заменить("что", "на_что")` — заменяет части текста.
* `текст.разделить(" ")` — разбивает строку по разделителю и возвращает список.

```son
пусть привет = "Привет Мир"
напечатай привет.верхний() # "ПРИВЕТ МИР"
напечатай привет.длина()   # 10
напечатай привет.содержит("Мир") # истина
```

### 3. Списки (Массивы)
Списки позволяют хранить множество элементов в одной переменной. Они создаются с помощью квадратных скобок `[]`.

```son
пусть инвентарь = ["Меч", "Зелье", "Щит"]
напечатай инвентарь[0] # "Меч"
```

Методы для работы со списками:
* `список.добавь(значение)` — добавляет элемент в конец.
* `список.удали(индекс)` — удаляет элемент по индексу. Если индекс не передан, удаляет последний элемент.
* `список.вставь(индекс, значение)` — вставляет элемент на указанную позицию.
* `список.содержит(значение)` — проверяет наличие элемента в списке.

```son
пусть числа = [1, 2, 3]
числа.добавь(4)
напечатай числа # Выведет [1, 2, 3, 4]
числа.удали(0)  
напечатай числа # Выведет [2, 3, 4]
```

## Преобразование типов
Часто нужно превратить число в текст (например, для вывода) или наоборот. Для этого есть встроенные функции:
* `строка(значение)` — превращает число или логическое значение в текст.
* `число("текст")` — превращает текст в число.
* `тип(переменная)` — возвращает название типа (`Number`, `String`, `Bool`, `List`, `Null`).

```son
пусть возраст = 20
# Нельзя просто сложить текст и число, нужно преобразовать:
напечатай "Мне " + строка(возраст) + " лет"

пусть ввод = "125.5"
пусть расчет = число(ввод) + 10
напечатай расчет # 135.5
```
