# Система и Файлы

Для написания полезных скриптов, взаимодействующих с операционной системой, используются модули `систем` и `файл`.

---

## Модуль `систем`

* `систем.выполнить("команда")`
Запускает системную команду (как если бы вы ввели её в cmd/bash) и возвращает её текстовый вывод.
```son
пусть результат = систем.выполнить("ping 8.8.8.8 -n 1")
напечатай результат
```

* `систем.выполнить_тихо("команда")`
Запускает системную команду, но не ждет её текстового вывода и ничего не возвращает. Полезно для запуска других программ в фоне.
```son
систем.выполнить_тихо("calc.exe") # Откроет стандартный калькулятор Windows
```

* `систем.список_файлов("папка")`
Возвращает список (массив) имен файлов и папок внутри указанной директории. Если передать `"."`, покажет текущую папку.
```son
пусть файлы = систем.список_файлов(".")
напечатай файлы
```

* `систем.рабочая_папка()`
Возвращает полный путь к папке, из которой был запущен скрипт.

* `систем.среда("имя_переменной")`
Получает значение переменной среды ОС (Environment Variable).
```son
напечатай систем.среда("PATH")
```

* `систем.платформа()`
Возвращает название вашей операционной системы (например, `"windows"` или `"linux"`).

* `систем.выйти(код)`
Немедленно завершает программу с указанным кодом выхода. Код `0` означает успешное завершение.
```son
систем.выйти(0)
```

---

## Модуль `файл`

* `файл.читать("имя_файла.txt")`
Считывает всё содержимое файла и возвращает как строку. Если файла нет, вернет пустоту.
```son
пусть текст = файл.читать("настройки.txt")
напечатай текст
```

* `файл.записать("имя_файла.txt", "Текст")`
Перезаписывает файл указанным текстом. Если файла нет — создаст его.
```son
файл.записать("сохранение.txt", "Уровень=5")
```

* `файл.добавить("имя_файла.txt", "Текст")`
Дописывает текст в конец файла, не удаляя старое содержимое.
```son
файл.добавить("логи.txt", "Программа запущена\n")
```
