# 🚀 SonLang (Язык программирования Son)

[![Платформа](https://img.shields.io/badge/платформа-Windows-blue.svg)](https://github.com/)
[![Лицензия](https://img.shields.io/badge/лицензия-MIT-green.svg)](https://github.com/)
[![Компилятор](https://img.shields.io/badge/движок-Rust-orange.svg)](https://github.com/)

**SonLang** — это простой, интуитивно понятный интерпретируемый язык программирования с полностью русскоязычным синтаксисом. Создан для быстрого обучения программированию, легкого системного администрирования, автоматизации рутины и создания полноценных 2D-игр и графических приложений.

Главный принцип SonLang: **«Код должен читаться как обычный русский текст»**.

---

## ✨ Ключевые фичи языка

*   **Русский синтаксис (Кириллица)**: Забудьте про английский барьер — используйте `пусть`, `если`, `пока`, `функция`, `импорт`.
*   **Идеальный вид**: Никаких точек с запятой `;` на конце строк и лишних скобок.
*   **Простые блоки кода**: Блоки начинаются с двоеточия `:` и закрываются словом `конец`.
*   **Динамическая типизация**: Интерпретатор сам определит, где число, строка, логический флаг или список.
*   **Встроенная 2D Графика и GUI**: Создание графических окон, кнопок и меток без установки внешних библиотек.
*   **Модули как в Python**: Легкий импорт файлов через `импорт` с полной поддержкой взаимного импорта (circular imports).
*   **Невероятная скорость**: Ядро написано на Rust, что гарантирует мгновенный запуск и выполнение миллионов математических операций в секунду.

## 💻 Пример программы (Стильная GUI кнопка-счетчик)

```son
модуль графика
модуль gui
модуль время

# Импортируем наш собственный дизайн-модуль!
импорт дизайн

графика.окно(400, 300, "Панель SonLang", истина)
gui.инициализировать()

пусть кликов = 0

пока графика.закрыто() == ложь:
    графика.очистить("#1A1A2E") # Темная премиум-тема
    
    дизайн.заголовок(20, 20, "ГЛАВНЫЙ ЭКРАН", "#00FF88")
    дизайн.кнопка("btn_main", 50, 100, 300, 50, "НАЖМИ НА МЕНЯ", "#E94560")
    
    gui.метка(50, 200, "Счетчик кликов: " + строка(кликов), "#FFFFFF")
    
    gui.нарисовать()
    графика.обновить()
    
    если gui.нажата("btn_main"):
        кликов = кликов + 1
    конец
    
    время.пауза(16) # Стабильные 60 кадров/сек
конец
```

## 🛠 Быстрая установка и запуск

### Вариант 1: Через готовый установщик (Рекомендуется)
1. Скачайте файл **`SonLang_Installer.msi`** из релизов репозитория.
2. Запустите его и следуйте простым инструкциям мастера установки.
3. Готово! Теперь вы можете запускать любой `.son` файл двойным кликом мыши в Windows!

### Вариант 2: Запуск из консоли
Вы можете вызвать интерпретатор напрямую в командной строке:
```cmd
son имя_скрипта.son
```
